# 个人知识库 Skill 使用指南

## 📌 什么是这个 Skill？

**Personal Knowledge Base Skill** 是一个专门为您的 Obsidian + GitHub 知识库系统设计的 AI 技能包。

### 核心功能

🔍 **智能检索**：当您提问时，AI 会优先搜索您的本地知识库  
📝 **自动记录**：对话中的想法和知识自动整理成笔记  
🔄 **无缝同步**：自动提交并推送到 GitHub  
💡 **知识闭环**：让您积累的知识真正为您工作

---

## 🚀 如何使用

### 方法 1：在新对话中加载（推荐）

每次开始新对话时，说：

```
加载 personal-kb skill
```

或

```
使用个人知识库技能
```

AI 就会自动加载这个 skill，并开始优先检索您的本地知识库。

### 方法 2：在当前对话中使用

如果已经在对话中，可以随时说：

```
请使用 personal-kb skill
```

## 🎯 工作原理

### 检索优先级

```
用户提问
    ↓
1️⃣ 搜索本地 Obsidian 知识库 ← 优先！
    ├─ 找到 → 基于笔记回答
    └─ 未找到 → 使用其他知识源
        ├─ MCP 知识库
        ├─ 互联网搜索
        └─ 通用知识
```

### 自动触发条件

Skill 会在以下情况自动工作：

✅ 您提出概念性问题："什么是..."  
✅ 您查询方法："如何..."  
✅ 您想回顾笔记："我之前记录了..."  
✅ 您分享想法或学习内容  
✅ 您明确要求记录内容

❌ 不会触发的情况：

- 简单对话（"你好"、"谢谢"）
- 纯操作请求（"运行命令"）
- 实时信息查询（"天气如何"）

## ❓ 常见问题

### Q：每次都要手动加载吗？

**A**：是的，由于 Skill 是动态加载的，每次新对话都需要加载。但您可以在第一句话中就说明：

```
加载 personal-kb skill，然后帮我...
```

### Q：如何知道 Skill 是否生效？

**A**：当您提问时，AI 会说"让我先检索您的本地知识库..."，这表示 Skill 已激活。

### Q：能在手机上使用吗？

**A**：可以！只要您的对话界面支持，都可以加载和使用这个 Skill。笔记会同步到 GitHub，可以在任何设备上查看。

### Q：Skill 会搜索所有问题吗？

**A**：不会。Skill 只在需要信息查询的知识类问题时才搜索，避免不必要的操作。

## 🎊 优势总结

### 相比 Memory 的优势

| 特性 | Memory | Skill |
|------|--------|-------|
| **持久性** | ❌ 仅当前会话 | ✅ 可重复加载 |
